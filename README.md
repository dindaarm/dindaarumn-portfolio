# Portfolio Landing Page - Dinda Arum Nugroho

Sebuah *landing page* portofolio pribadi yang dirancang secara elegan untuk menampilkan profil, keahlian, dan proyek-proyek unggulan. Proyek ini dibuat khusus untuk menonjolkan persona seorang **Data Analyst** dengan antarmuka yang profesional dan dinamis.

## 🌟 Fitur Utama

- **Desain Premium Dark Mode**: Menggunakan skema warna gelap (*Slate*) dengan aksen gradasi biru-ke-cyan yang nyaman di mata dan memberikan kesan modern.
- **Efek Glassmorphism**: Penggunaan efek kaca buram (*blur*) pada navigasi dan kartu konten (*cards*).
- **Animasi Scroll (Reveal)**: Elemen-elemen visual akan muncul secara bertahap saat Anda menggulir (*scroll*) halaman ke bawah, memberikan pengalaman pengguna yang interaktif.
- **Responsif Sepenuhnya**: Tata letak yang dioptimalkan dengan CSS Flexbox dan Grid agar tampil sempurna di layar desktop, tablet, maupun *smartphone*.
- **Simulasi Formulir Kontak**: Formulir interaktif dengan umpan balik (*feedback*) berupa tombol *loading* animasi.

## 🛠️ Teknologi yang Digunakan

Proyek ini dibangun murni menggunakan tumpukan web dasar demi menjaga performa yang ringan dan cepat tanpa ketergantungan eksternal:
- **HTML5**: Untuk menyusun struktur semantik web.
- **CSS3 (Vanilla)**: Digunakan untuk gaya, tata letak, variabel *root*, dan animasi transisi.
- **JavaScript (Vanilla)**: Digunakan untuk manipulasi DOM (seperti *sticky navbar*) dan penggunaan *Intersection Observer API* untuk mendeteksi *scroll*.

## 🚀 Cara Menjalankan

Tidak diperlukan proses *build* atau instalasi *library* sama sekali. Anda dapat menjalankannya secara instan:

1. Pastikan semua file berada dalam satu direktori yang sama.
2. Cari file `index.html` di dalam *File Explorer* Anda.
3. Klik ganda (`double-click`) file `index.html` tersebut.
4. Halaman akan langsung terbuka di peramban web (*browser*) default Anda seperti Google Chrome, Microsoft Edge, atau Firefox.

## 📂 Struktur File

```text
 ┣ 📜 index.html    # Dokumen utama berisi struktur konten dan profil
 ┣ 📜 styles.css    # Berisi semua deklarasi gaya dan animasi (Dark Theme)
 ┣ 📜 script.js     # Logika interaktif untuk efek gulir dan UI
 ┗ 📜 README.md     # Ikhtisar dokumentasi proyek ini
```
