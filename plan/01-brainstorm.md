# Fase 1: Brainstorming (Ideasi)

## Tujuan Proyek
Membuat sebuah *landing page* portofolio pribadi yang profesional, menarik, dan modern untuk **Dinda Arum Nugroho**, seorang Data Analyst.

## Target Audiens
- Perekrut kerja (*HR / Recruiters*)
- Manajer Proyek (*Project Managers*)
- Klien potensial yang membutuhkan jasa analisis data.

## Konsep Desain
1. **Tema Utama**: Profesional, analitis, namun tetap kreatif.
2. **Skema Warna**: Awalnya direncanakan menggunakan tema terang (*Light Mode*), namun direvisi menjadi **Dark Mode** menggunakan palet *Slate* (abu-abu kebiruan gelap) untuk memberikan kesan teknologi yang modern, elegan, dan *premium*.
3. **Gaya Visual (*Aesthetics*)**:
   - Penggunaan *Glassmorphism* (efek kaca transparan berembun) untuk memberikan kedalaman pada kartu konten (*cards*).
   - Aksentuasi warna gradasi cerah (biru ke cyan) untuk memandu mata audiens pada elemen penting (teks judul, tombol).
   - Animasi mikro (*micro-interactions*) agar halaman terasa "hidup".

## Kebutuhan Fitur
- **Hero Section**: Perkenalan cepat, peran (Data Analyst), dan foto diri.
- **Tentang Saya**: Ringkasan latar belakang dan daftar keterampilan (*tools*).
- **Portofolio/Proyek**: Etalase hasil kerja (visualisasi data, model prediksi).
- **Kontak**: Formulir simpel untuk berkolaborasi dan tautan ke media sosial profesional (LinkedIn, GitHub, Email).
